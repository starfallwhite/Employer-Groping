# Employer-Groping
STL案例2，员工分组
README文件

需求：
* 公司今天招聘了10个员工（ABCDEFGHIJ），10名员工进入公司之后，需要指派员工在那个部门工作
* 员工信息有: 姓名  工资组成；部门分为：策划、美术、研发
* 随机给10名员工分配部门和工资
* 通过multimap进行信息的插入  key(部门编号) value(员工)
* 分部门显示员工信息

实现：
员工类：姓名，工资
员工容器vector<employer>;

员工部门表<部门，员工>；随机生成部门，将员工与随机生成的部门插入表中
分组结果插入multimap中：

随机分配部门和工资：创建员工时随机生成工资

显示员工信息
使用员工部门表的迭代器遍历表，按部门打印员工信息
